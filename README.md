🎬 Netflix Data Visualization Project

📌 Overview

This project performs Exploratory Data Analysis (EDA) and Data Visualization on the Netflix Titles Dataset using Python, Pandas, and Matplotlib.

The objective is to uncover trends, patterns, and insights from Netflix’s collection of Movies and TV Shows through various visualizations.

⸻

🚀 Features

* Data Cleaning & Preprocessing
* Missing Value Handling
* Exploratory Data Analysis (EDA)
* Bar Charts
* Pie Charts
* Line Charts
* Histogram Analysis
* Subplots Dashboard
* High-Quality Figure Exporting

⸻

📊 Visualizations Included

1. Movies vs TV Shows Distribution

* Compare the number of Movies and TV Shows available on Netflix.

2. Top 10 Countries

* Identify countries producing the most Netflix content.

3. Content Rating Distribution

* Analyze audience ratings such as TV-MA, PG, TV-14, etc.

4. Netflix Content Growth Over Time

* Visualize how Netflix’s library expanded over the years.

5. Top 10 Genres

* Discover the most common genres on Netflix.

6. Top Directors

* Identify directors with the highest number of titles.

7. Dashboard Visualization

* Multiple charts combined using Matplotlib Subplots.

📈 Key Insights

* Movies significantly outnumber TV Shows on Netflix.
* The United States contributes the largest amount of content.
* TV-MA is among the most common content ratings.
* Netflix experienced rapid content growth after 2015.
* Drama and International Movies are among the most popular genres.
